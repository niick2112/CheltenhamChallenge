Rules

1. POINTS

1.1 Points are calculated based on official start price odds and multipliers against position finished.

1.2 Each race you are staking one

1.3 Multiplier is applied against decimal odds e.g. 5/1 = 6.0. 7/2 = 4.5.

1.4 Multipliers as follows:

  • 1st place x 1
  • 2nd place x 0.4
  • 3rd place x 0.2
  • 4th place x 0.1

1.5 An example of points base on a 10/1(11.0 in decimal format) horse being selected and placing.

  • 1st = 11
  • 2nd = 4.4
  • 3rd = 2.2
  • 1st x 1.1

2. PLACES

2.1 In a race with 4 horses or less only 1st place will be paid.

2.2 In a race with 5-7 horses 1st and 2nd place will be paid.

2.3 In a race with 8-15 horses 1st, 2nd and 3rd will be paid.

2.4 In a race with 16 plus horses 1st, 2nd, 3rd & 4th will be paid.

3. DEAD HEAT

3.1 If there is a dead heat for 1st place. The multiplier for 1st place will be applied to horses involved in dead heat. The third horse over the line will have 3rd place multiplier applied.

3.2 If there is a dead heat for 2nd place. The multiplier for 2nd place will be applies to horses involved in dead heat. 1st place will be applied to 1st place.

3.3 If there is a dead heat for 3rd place or 4th place, where applicable, then 1st and 2nd place will be applied as expected and 3rd or 4th place will be applied to horses involved in the dead heat.

4. NON RUNNER

4.1 If you discover one of your entries has been declared a non runner before the cut off then you may message one of the event organisers and change your selection.

4.2 Changes must be messaged via text or WhatsApp so a time of selection change is provided.

4.3 If your horse is declared a non runner after the cut off then the start price favourite will become your selection.

4.4 If there is a joint start price favourite then alphabetical order applies.

5. CUT OFF / ENTRIES

5.1 Cut off is 1pm sharp.

5.2 Failure to enter before 1pm but before the first race will incur a 3 point deduction.

5.3 Failure to enter before the first race will result in you only being able to score points from / including race 4 onwards for that day. Rule 5.2 will also apply.

5.4 Entries are to be made via the link provided each day.

5.5 Entries must have your name written correctly.

5.6 Entries must have horse name spelt correctly.

5.7 Entry fee for the challenge is £15. This must be transferred to Stuart on/by the Monday of Cheltenham festival. No money no entry.

6 SCORE SHEET

6.1 The scores will be available to view from 1pm each day.

7 WINNINGS

7.1 There will be a daily prize for the player with the highest score each day.

7.2 There will be an overall table too. The overall table will pay 3 places.

7.3 Prize money will be confirmed once all money has been paid.

7.4 Prize money will be paid out at the end of the festival.

8 QUERIES

8.1 There will not be any disputes or queries because these rules are full proof and now written down!

Bonus pot

As mentioned previously we will be running a group bet too, in an attempt to make everyone a winner! Very pc!

Each day the 4 most popular picks will become a 50p lucky 15. If there is a tie on the horses the horse with the highest odds at the time we lay the bet will be selected. If all the odds are double digits then the bet will become a 25p ew bet.

If there isn't 4 standout horses the leading entrant choice will be selected from the first available race. 2nd place entrant from second available race and so on.

On day one if there isn't 4 standout horses the favourite from the first available race will be selected. 2nd favourite from the second available race and so on.

For those of you that don't know, the joy of a lucky 15 is you get a return if only one horse wins.
